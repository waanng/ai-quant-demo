机器学习选股模型 — 季度 Top 30 策略¶

目标：用 A 股财务因子（估值 + 成长共 19 个）训练机器学习模型，预测股票下季度收益率 Next_Ret，每个季度选取预测得分最高的 30 只构建组合，对比 等权 与 预测值加权 两种配置方式的效果。

流程：数据加载 → 预处理（去极值 + 标准化 + 时间切分 + 双标签）→ 四模型训练 → Top 30 选股 → 等权 vs 加权对比 → 策略评估（净值曲线 / 指标表 / 分组检验 / IC）。

数据：model_data.csv（39,616 行 × 22 列，10 个季度 2020Q1–2022Q2） 训练集：2020Q1–2021Q2（6 季）｜ 测试集：2021Q3–2022Q2（4 季）

4. 数据预处理与特征工程¶

  1. 缺失值：中位数填充
  2. 去极值（MAD）：用训练集的中位数 ± 5×MAD 截断极端值，避免个别异常值压垮模型
  3. 标准化：StandardScaler 仅用训练集 fit，再 transform 测试集（防数据泄露）
  4. 时间切分：严格按季度，训练集在前、测试集在后
  5. 双标签：
    • 回归标签：Next_Ret 原值（线性回归 / 决策树 / 随机森林）
    • 分类标签：季度内 Next_Ret 排名前 30% → 1，否则 0（逻辑回归，用预测概率排序选股）
In [6]:
train_quarters = ['2020Q1','2020Q2','2020Q3','2020Q4','2021Q1','2021Q2']
test_quarters  = ['2021Q3','2021Q4','2022Q1','2022Q2']
train_mask = raw['Quarter'].isin(train_quarters)
test_mask  = raw['Quarter'].isin(test_quarters)

# 1) 缺失值填充
imputer = SimpleImputer(strategy='median')
X_imputed = pd.DataFrame(
    imputer.fit_transform(raw[feature_cols]),
    columns=feature_cols, index=raw.index)

# 2) MAD 去极值（用训练集估计参数，应用到全量）
mad_params = {}
for c in feature_cols:
    med = X_imputed.loc[train_mask, c].median()
    mad = (X_imputed.loc[train_mask, c] - med).abs().median()
    mad_params[c] = (med, mad)
def clip_mad(df):
    df = df.copy()
    for c in feature_cols:
        med, mad = mad_params[c]
        df[c] = df[c].clip(med - 5*mad, med + 5*mad)
    return df
X_clip = clip_mad(X_imputed)

# 3) 标准化
scaler = StandardScaler()
X_train = pd.DataFrame(scaler.fit_transform(X_clip.loc[train_mask]), columns=feature_cols)
X_test  = pd.DataFrame(scaler.transform(X_clip.loc[test_mask]),  columns=feature_cols)

# 4) 标签
y_train_reg = raw.loc[train_mask, 'Next_Ret'].values
y_test_reg  = raw.loc[test_mask,  'Next_Ret'].values
def make_cls(df):
    return df.groupby('Quarter')['Next_Ret'].transform(lambda s: (s >= s.quantile(0.7)).astype(int))
y_train_cls = make_cls(raw.loc[train_mask]).values
y_test_cls  = make_cls(raw.loc[test_mask]).values

# 测试集基础表（选股用）
test_df = raw.loc[test_mask, ['Date','Quarter','Code','Next_Ret']].reset_index(drop=True)

print('训练样本数:', len(X_train), '| 测试样本数:', len(X_test))
print('训练正类占比(Top30%):', round(y_train_cls.mean(),3), '| 测试:', round(y_test_cls.mean(),3))

9. Top 30 选股策略构建¶

对测试集每个季度：按模型预测得分降序排序，取前 30 只，分别构建两种组合：

  • 等权组合：每只权重 = 1/30
  • 预测值加权组合：权重 = 预测得分（负值置 0）归一化，得分高的多配

并构建"等权全市场"基准作对比。

In [11]:
def build_portfolio(ts, top_n=30):
    rows = []
    for q in sorted(ts['Quarter'].unique()):
        sub = ts[ts['Quarter']==q].copy().sort_values('score', ascending=False).head(top_n)
        eq = sub['Next_Ret'].mean()
        w = sub['score'].clip(lower=0)
        w = w / w.sum() if w.sum() > 0 else pd.Series(1/len(sub), index=sub.index)
        pw = (w * sub['Next_Ret']).sum()
        rows.append({'Quarter': q, 'equal_ret': eq, 'pred_ret': pw, 'n': len(sub)})
    return pd.DataFrame(rows)

portfolios = {}
for name, ts in model_test_scores.items():
    pf = build_portfolio(ts)
    pf['cum_equal'] = (1 + pf['equal_ret']).cumprod() - 1
    pf['cum_pred']  = (1 + pf['pred_ret']).cumprod() - 1
    portfolios[name] = pf
    print(f"{name:6s} | 等权累计 {pf['cum_equal'].iloc[-1]:+.3f} | 预测加权累计 {pf['cum_pred'].iloc[-1]:+.3f}")

bench = test_df.groupby('Quarter')['Next_Ret'].mean().rename('bench_ret').reset_index()
bench['cum_bench'] = (1 + bench['bench_ret']).cumprod() - 1
print('基准    | 等权全市场累计', f"{bench['cum_bench'].iloc[-1]:+.3f}")

def metrics(ret, bench_ret):
    cum = (1 + ret).cumprod() - 1
    total = cum.iloc[-1]
    ann = (1 + total) ** (4/len(ret)) - 1
    vol = ret.std() * np.sqrt(4)
    sharpe = (ann - 0.02) / vol if vol > 0 else np.nan
    nav = (1 + ret).cumprod()
    mdd = (nav / nav.cummax() - 1).min()
    win = (ret > bench_ret).mean()
    return dict(累计收益=total, 年化收益=ann, 年化波动=vol, 夏普=sharpe, 最大回撤=mdd, 胜率=win)

rows = []
for name, pf in portfolios.items():
    me = metrics(pf['equal_ret'], bench['bench_ret']); me.update({'模型':name,'加权方式':'等权'}); rows.append(me)
    mp = metrics(pf['pred_ret'],  bench['bench_ret']); mp.update({'模型':name,'加权方式':'预测加权'}); rows.append(mp)

# IC 分析：每季度预测分与实际收益的 Spearman 相关
ic_rows = []
for name, ts in model_test_scores.items():
    ic_list = []
    for q in sorted(ts['Quarter'].unique()):
        d = ts[ts['Quarter']==q]
        ic_list.append(d['score'].corr(d['Next_Ret'], method='spearman'))
    ic = pd.Series(ic_list, index=sorted(ts['Quarter'].unique()))
    ic_rows.append({'模型':name, '平均IC':ic.mean(), 'IC_IR':ic.mean()/ic.std()})
ic_df = pd.DataFrame(ic_rows).round(4)
print('===== IC 分析 =====')

11. 结论与选型建议¶

  • 净值曲线：四种模型的「等权」与「预测加权」组合均画在同一张图，黑色基准线为全市场等权。
  • 指标表：累计收益、年化、夏普、最大回撤、胜率一目了然；通常随机森林/逻辑回归类非线性模型在选股上优于纯线性回归。
  • 分组检验：若第 5 组（预测最高）系统性高于第 1 组（预测最低），说明模型具备真实选股能力，而非偶然。
  • IC：平均 IC 越接近 1、IC_IR 越大，模型排序能力越强。
  • 等权 vs 加权：预测值加权在模型排序能力强时通常优于等权；若模型 IC 偏低，加权可能放大噪声，反而不如等权稳健。
