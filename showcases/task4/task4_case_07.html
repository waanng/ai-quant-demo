第一章：策略概述

1.1 海龟交易策略核心思想

海龟交易法则由 Richard Dennis 与 William Eckhardt 在 1983 年提出，是一套完整的趋势跟踪系统。其核心思想是：

  • 顺势而为：不预测市场方向，只在价格突破近期高点时建立多头仓位，跟随趋势。
  • 波动率定价：用 ATR 衡量市场波动，以此决定仓位大小与止损距离。
  • 机械执行：所有规则量化、可编程，避免人为情绪干扰。
  • 风险控制：通过 2×ATR 跟踪止损与金字塔加仓，限制单笔损失并放大盈利。
关键优势

1) 规则明确，可完全自动化；2) 在强趋势行情中能捕获大额利润；3) 通过 ATR 动态调整仓位，波动大时自动减仓；4) 分散化交易，不押注单一标的。

1.2 核心概念

高低点通道（Donchian Channel）：由过去 N 个交易日的最高价与最低价构成的通道。上轨代表阻力位，下轨代表支撑位。

上轨 = max(high[t-N+1], ..., high[t])
下轨 = min(low[t-N+1], ..., low[t])
突破上轨 → 买入信号；跌破下轨 → 卖出/退出信号

平均真实波幅（ATR）：衡量市场波动率的指标。真实波幅 TR 取当日最高价-最低价、|最高价-昨收|、|最低价-昨收| 三者最大。

TR = max(high-low, |high-pre_close|, |low-pre_close|)
ATR = TR 的 N 日移动平均

止损条件：本报告采用 ATR-based 跟踪止损。入场后，止损价随价格上涨而上移；若价格从最高点回落 2×ATR 或跌破下轨，则清仓。

止损价 = max(入场价 - 2×ATR, 当前价 - 2×ATR)
加仓价 = 上一次入场价 + 0.5×ATR；最多 4 单位

第六章：总结与应用心得

6.1 策略优缺点

  • 优点：规则机械、可完全量化；在强趋势中能够捕捉主升段；ATR 仓位管理使高波动时风险降低。
  • 缺点：震荡市和下跌趋势中会产生连续亏损或空仓；信号滞后，入场价格往往远离底部；需要较大样本才能体现优势。

6.4 改进方向

  • 使用更长的历史样本和更多标的进行样本外验证。
