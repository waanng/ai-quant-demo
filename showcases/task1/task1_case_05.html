分析二 ｜ 技术面分析（均线 / MACD / KDJ / RSI / BOLL）

技术分析通过数学统计方法，从历史价格和成交量数据中提取趋势、动量、波动性等特征，辅助判断未来走势。 以下为当前各项技术指标的综合解读：

1. 均线系统 (MA5 / MA10 / MA20 / MA60)

移动平均线(MA)反映一段时间内的平均持仓成本。短期均线在上穿长期均线时形成"金叉"(看多)， 下穿时形成"死叉"(看空)。多头排列(MA5>MA10>MA20>MA60)是最强势的技术形态之一。

2. MACD 指标 (12, 26, 9)

MACD由DIF线(快线)、DEA线(慢线)和柱状图组成。DIF上穿DEA为金叉(买入信号)， 下穿为死叉(卖出信号)。柱状图由负转正表示多头动能开始增强。MACD适合捕捉中期趋势转折。

3. KDJ 指标 (9, 3, 3)

KDJ是随机震荡指标，K线与D线的交叉提供短线买卖信号。J值>80为超买区(注意回调)， J值<20为超卖区(关注反弹)。KDJ在震荡市中效果较好，趋势市中容易钝化。

4. RSI 相对强弱指标 (6, 12, 24)

RSI衡量价格上涨幅度在总波动中的占比。RSI>70为超买，RSI<30为超卖。 RSI6反应最灵敏，RSI24最平滑。多条RSI线同时进入超买/超卖区，信号更可靠。

5. 布林带 BOLL (20, 2)

布林带由中轨(MA20)、上轨(+2σ)、下轨(-2σ)组成。股价触及上轨可能超买，触及下轨可能超卖。 布林带开口收窄预示波动率降低，可能即将迎来方向性突破。
