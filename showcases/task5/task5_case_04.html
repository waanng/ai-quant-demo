一、分类型机器学习算法解释¶

分类算法用于预测样本属于哪一个类别。在量化交易中，分类模型可以被用来判断“未来收益是否为正”“是否出现买入信号”“股票是否进入高风险状态”等二分类或多分类问题。本任务重点了解逻辑回归、决策树和随机森林三类常见分类算法。

1. 逻辑回归 Logistic Regression¶

逻辑回归虽然名字里有“回归”，但它常用于分类问题。它通过特征变量的线性组合计算一个概率值，再根据概率是否超过阈值判断类别。例如在二分类中，模型可以输出某个样本属于正类的概率。如果概率大于 0.5，就预测为正类；否则预测为负类。

逻辑回归的优点是模型结构简单、训练速度快、结果相对容易解释。每个特征对应一个系数，系数正负可以帮助理解该特征对分类结果的影响方向。缺点是它主要刻画线性关系，如果数据中存在复杂非线性结构，单独使用逻辑回归可能表现有限。

2. 决策树 Decision Tree¶

决策树是一种类似“规则判断”的分类模型。它会根据特征的取值不断进行分裂，例如“某个指标是否大于某个阈值”，最终把样本分到不同类别。决策树的结构类似流程图，因此可解释性较强。

决策树的优点是容易理解，能够处理非线性关系，也不需要对特征做复杂标准化。缺点是单棵树容易过拟合，也就是在训练集上表现很好，但在测试集上泛化能力不足。因此实际使用时常通过限制树深度、设置最小样本数等方式控制复杂度。

3. 随机森林 Random Forest¶

随机森林是一种集成学习模型，它由多棵决策树组成。每棵树使用部分样本和部分特征进行训练，最后通过投票或平均概率得到最终分类结果。

随机森林的优点是比单棵决策树更稳定，抗过拟合能力更强，能够处理非线性关系，并且可以输出特征重要性。缺点是模型比单棵树更复杂，解释性稍弱，训练和预测成本也更高。

在量化交易中，随机森林可以用于综合多个技术指标、财务指标和行情特征，判断未来收益方向或风险状态。

二、机器学习模型评价指标解释¶

分类模型不能只看预测是否“看起来合理”，还需要通过评价指标进行检验。本任务重点解释混淆矩阵、ROC 曲线和 AUC。

1. 混淆矩阵 Confusion Matrix¶

混淆矩阵用于比较模型预测类别和真实类别之间的关系。二分类混淆矩阵通常包含四个部分：

  • TP 真正例：真实为正类，模型也预测为正类。
  • TN 真负例：真实为负类，模型也预测为负类。
  • FP 假正例：真实为负类，但模型预测为正类。
  • FN 假负例：真实为正类，但模型预测为负类。

混淆矩阵可以帮助观察模型错在哪里。例如在金融场景中，如果把“未来上涨”作为正类，那么假正例可能表示模型误判上涨，导致错误买入；假负例可能表示模型错过上涨机会。

2. ROC 曲线 Receiver Operating Characteristic Curve¶

ROC 曲线用于评价二分类模型在不同分类阈值下的表现。横轴是假正率 FPR，纵轴是真正率 TPR。

  • TPR = TP / (TP + FN)，表示正类样本中被正确识别出来的比例，也叫召回率。
  • FPR = FP / (FP + TN)，表示负类样本中被错误预测为正类的比例。

当分类阈值变化时，TPR 和 FPR 会一起变化。ROC 曲线越靠近左上角，说明模型越能在较低误判率下识别更多正类，模型区分能力越强。

3. AUC Area Under Curve¶

AUC 是 ROC 曲线下方的面积，用于衡量模型整体区分正负样本的能力。AUC 的取值通常在 0 到 1 之间：

  • AUC = 0.5：模型接近随机猜测。
  • AUC 越接近 1：模型区分能力越强。
  • AUC 小于 0.5：模型可能方向反了，预测效果比随机还差。

在量化交易中，AUC 可以用来判断模型是否能够有效区分“未来上涨”和“未来下跌”等类别。不过 AUC 高并不等于策略一定赚钱，还需要结合交易成本、仓位管理、回撤和实盘约束进一步评估。

# 在测试集上评估模型，计算 AUC 等指标。
evaluation_rows = []
prediction_results = {}

for model_name, model in models.items():
    y_pred = model.predict(X_test)
    y_score = model.predict_proba(X_test)[:, 1]
    auc_value = roc_auc_score(y_test, y_score)
    fpr, tpr, thresholds = roc_curve(y_test, y_score)

    evaluation_rows.append({
        "模型": model_name,
        "准确率 Accuracy": accuracy_score(y_test, y_pred),
        "精确率 Precision": precision_score(y_test, y_pred),
        "召回率 Recall": recall_score(y_test, y_pred),
        "F1值": f1_score(y_test, y_pred),
        "AUC": auc_value,
    })

    prediction_results[model_name] = {
        "y_pred": y_pred,
        "y_score": y_score,
        "fpr": fpr,
        "tpr": tpr,
        "thresholds": thresholds,
        "confusion_matrix": confusion_matrix(y_test, y_pred, labels=[0, 1]),
    }

metrics_df = pd.DataFrame(evaluation_rows).sort_values("AUC", ascending=False).reset_index(drop=True)
metrics_display = metrics_df.copy()
for col in ["准确率 Accuracy", "精确率 Precision", "召回率 Recall", "F1值", "AUC"]:
    metrics_display[col] = metrics_display[col].map(lambda value: f"{value:.4f}")

metrics_path = task5_dir / "classification_model_metrics.csv"
metrics_df.to_csv(metrics_path, index=False, encoding="utf-8-sig")

print(f"模型评估指标已保存至：{metrics_path}")
display(metrics_display)
