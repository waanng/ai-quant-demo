# RSI 计算公式 (Wilder 平滑法, 周期 N=14) avg_gain = 前日avg_gain × (N-1)/N + 今日涨幅 × 1/N # 递推平滑 avg_loss = 前日avg_loss × (N-1)/N + 今日跌幅 × 1/N RS = avg_gain / avg_loss RSI = 100 - 100 / (1 + RS)

RSI（Relative Strength Index）衡量价格"涨"的动能强度，值域 0~100。计算方式是 14 日内平均涨幅与平均跌幅的比值（Wilder 平滑法），再映射到 0~100。

判读规则：

  • RSI > 70：超买区，价格上涨过快，可能面临回调
  • RSI < 30：超卖区，价格下跌过猛，可能出现反弹
  • RSI ≈ 50：多空平衡，趋势不明

中芯国际 3 年数据中，RSI 最高冲到 92.49（极度超买），最低跌至 20.37（极度超卖）。超买天数 42 天、超卖天数 26 天，上行趋势占主导。当前 RSI = 50.41，处于中性区间——上涨动能正在衰减但尚未反转。

注意：RSI > 70 后可能"钝化"继续涨，< 30 也不意味着立刻反弹，需结合其他指标。

# MACD 计算公式 (快线=12, 慢线=26, 信号线=9) EMA_fast = close 的 12日指数加权移动平均 EMA_slow = close 的 26日指数加权移动平均 DIF = EMA_fast - EMA_slow # 差离值 DEA = DIF 的 9日EMA # 信号线 MACD柱 = DIF - DEA # 柱状图 (不乘2)

MACD（Moving Average Convergence Divergence）用快慢两条 EMA 线的差值捕捉趋势方向和转折。DIF = EMA(12) − EMA(26)，DEA = DIF 的 9 日 EMA。

判读规则：

  • 金叉：DIF 上穿 DEA，买入信号（柱状图由负转正）
  • 死叉：DIF 下穿 DEA，卖出信号（柱状图由正转负）
  • 柱状图：反映多空力量对比，红柱=多头占优，绿柱=空头占优
  • 背离：价格新高但DIF未新高，可能趋势反转

3 年内共发生 51次金叉/死叉信号（金叉25次，死叉26次）。当前 DIF = 5.53，DEA = 5.73，MACD 柱 = -0.20——柱体刚从正转负，意味着短期趋势出现拐头向下迹象，需警惕短线调整。

# 布林带计算公式 (周期=20, 标准差倍数=2) MID = close 的 20日简单移动平均 (SMA) STD = close 的 20日总体标准差 (ddof=0) UPPER = MID + 2 × STD LOWER = MID - 2 × STD 带宽 = (UPPER - LOWER) / MID

布林带（Bollinger Bands）以 20 日均线为中轨，上下各加减 2 倍标准差形成通道。带宽反映波动率：通道收窄说明市场趋于平静（可能酝酿突破），通道扩张说明波动加剧。

判读规则：

  • 触及上轨：短期超买，价格可能回落
  • 触及下轨：短期超卖，价格可能反弹
  • 带宽收窄：波动率降低，可能即将出现方向性突破
  • 中轨方向：中轨向上=中期趋势偏多，向下=偏空

中芯国际 3 年中触上轨 53天、触下轨 26天，上触频率是下触的两倍，印证了上涨主导格局。当前上轨 ¥162.33、中轨 ¥139.17、下轨 ¥116.02，价格 ¥140.31 位于中轨上方——处于偏强区间但距上轨尚有 15.7% 空间。

ATR (14) 平均真实波幅 Wilder 平滑
当前ATR: 9.53 历史最低: 0.94 历史最高: 9.78 ATR/价格: 6.8%
# ATR 计算公式 (Wilder 平滑法, 周期 N=14) TR = max( High - Low, # 当日波幅 |High - 前日Close|, # 跳空向上 |Low - 前日Close| # 跳空向下 ) ATR = 前日ATR × (N-1)/N + 今日TR × 1/N # Wilder 递推平滑

ATR（Average True Range）衡量价格的日内波动幅度，不同于收盘价波动率，ATR 考虑了跳空缺口。TR（真实波幅）取 H−L、|H−前日收盘|、|L−前日收盘| 三者的最大值，再用 Wilder 递推法做 14 日平滑。

ATR 不指示方向，只反映"波动有多大"。主要用途：

  • 止损设置：止损位 = 入场价 − 1.5~2 倍 ATR
  • 仓位管理：ATR 越大，单笔风险越大，应减少仓位
  • 波动率监测：ATR 飙升预示大行情将至
  • 趋势过滤：ATR 低位+价格横盘=市场犹豫期

中芯国际 ATR 范围从 0.94（极平静）到 9.78（剧烈波动），相差约 10 倍。当前 ATR = 9.53，接近历史最高值，ATR/价格 = 6.8%，意味着日均波动约 ¥9.5。适合设置止损位（如 1.5 倍 ATR ≈ ¥14.3 止损）。
